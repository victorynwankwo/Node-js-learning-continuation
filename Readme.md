### 🧩 Project Structure

- `server.js` – main app entry point
- `htttp.js` – working on server logic and route handling
- `routes/` – modular route definitions for root, subdir, and API endpoints
- `controllers/` – request handler logic to keep code clean
- `middleware/` – custom middleware for logging and error handling
- `config/` – configuration settings such as CORS options
- `public/` – static files, styles, images, and text resources
- `views/` – HTML pages for the site and nested folders
- `model/employee.json` – sample data for API learning

### 📌 Run the Project

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm run dev
   ```
3. Open the app in your browser and explore the routes
